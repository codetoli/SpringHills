# SpringHill School Website

Welcome to the SpringHill School website! This platform is designed to provide students, parents, and staff with a seamless and engaging experience. Below is a detailed overview of the features available on our website.

## Features

### 1. **Homepage**

- A visually appealing landing page with a dynamic banner and highlights of the school's latest updates.
- Quick navigation to important sections of the website.

### 2. **Notices Section**

- Displays official circulars and daily updates from the school management.
- Search functionality to find notices by keywords.
- Supports PDF and image previews for notices.
- Download option for all notices.

### 3. **Gallery**

- A dedicated section showcasing photos and videos from school events and activities.
- Organized into categories for easy browsing.

### 4. **Programs and Curriculum**

- Detailed information about the academic programs and extracurricular activities offered by the school.
- Highlights of unique programs and initiatives.

### 5. **Contact Page**

- A user-friendly contact form for inquiries and feedback.
- Displays the school's address, phone number, and email for direct communication.

### 6. **Enrollment Page**

- Step-by-step guide for new student enrollment.
- Online application form for prospective students.

### 7. **Admin Panel**

- Secure access for school administrators to manage content and updates.
- Tools for uploading notices, managing gallery content, and more.

### 8. **Responsive Design**

- Fully optimized for desktop, tablet, and mobile devices.
- Ensures a seamless user experience across all screen sizes.

### 9. **Search Functionality**

- Global search bar to quickly find content across the website.

### 10. **Latest News Section**

- Keeps users updated with the most recent happenings and announcements.

### 11. **Login System**

- Secure login for staff and administrators.
- Role-based access control for different user types.

### 12. **Footer with Social Media Links**

- Quick links to the school's social media profiles.
- Additional navigation links for easy access.

## Technologies Used

- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Firebase Firestore for real-time database and hosting
- **Icons**: Lucide React

## How to Run the Project

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Start the development server with `npm run dev`.
4. Open the website in your browser at `http://localhost:5173`.

We hope you enjoy exploring the SpringHill School website!
